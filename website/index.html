Accessibility Modes

Purpose-built mode behavior

Deaf Mode

Captions + tone cues stay visible, and the right panel runs MediaPipe gesture recognition from webcam input.

Blind Mode

Voice navigation, screen explanation, and audio feedback stay primary while gesture input is disabled.

Mode Switcher

Quick in-overlay switch persists mode preference and updates UI behavior immediately.

Gesture Input

On-device hand tracking in Deaf mode

Gesture Panel
  • Live webcam feed with landmark overlay and gesture confidence display.
  • Confidence thresholding (70%+) to reduce accidental command triggers.
  • Two-hand detection support with stable panel status updates.
  • Mapped command gestures: thumbs up/down, open palm, and pointing up.
  • Gesture and speech commands share the same central routing pipeline.
